🧑‍💻 Sui Vibe Hackathon

图片
Build on Sui · Build the Vibe



Sui Vibe Hackathon 是一场面向全球 Builder 的线上黑客松活动，由 HOH × Sui 联合主办，并获得 Cetus、Stablelayer 等生态伙伴支持。



同时，本次活动也得到了来自多个开发者社区及高校链协的推广支持，包括 OpenBuild、Coset、Cyclens、去探索、Vibe Friends、HerStory 及 Bermudao，共同推动更多开发者进入 Sui 生态、参与构建。


它不仅是一场编程比赛，更是一场围绕产品、创意与 Vibe的共创实验。我们鼓励开发者基于 Sui 公链与 Move 语言，构建真正可运行、可体验、具备产品完整度的应用。
